# 🧬 BioFlow — Ваш автономный биохакинг-трекер

**BioFlow** — это полностью локальное, приватное мобильное приложение для мониторинга физиологических показателей, ведения дневника состояния и отслеживания привычек. Приложение создано для тех, кто хочет экономить силы и время на пути к лучшей версии себя, сохраняя 100% конфиденциальность своих данных.

---

## 🚀 Основные возможности версии v1.0.3

* 📊 **Колесо жизненного баланса:** Оценка 8 ключевых сфер жизни с наглядной визуализацией на Radar Chart.
* ⏱️ **Еженедельный трекер привычек (Habit Sprint):** Удобная сетка привычек на текущую неделю с возможностью закрытия прошлых дней.
* 🔔 **Умные физиологические уведомления:** Анализ трендов показателей (сна, веса, давления) с точной синхронизацией напоминаний по вашему часовому поясу.
* 📅 **Лента времени (Timeline):** Объединенная история ваших биометрических показателей, дневников благодарности, фокусов дня и ярких моментов.
* 🔒 **Офлайн-манифест:** Никаких серверов, облаков и трекеров аналитики. База данных Isar хранится исключительно на вашем устройстве в зашифрованной песочнице.

## 📲 Как установить приложение на Android (в один клик)

1. Перейдите в раздел **[Releases](https://github.com/Kir0029/BioFlow/releases/latest)** (Релизы) на этой странице.
2. Скачайте файл **`app-release.apk`**.
3. Откройте скачанный файл на вашем смартфоне Android.
4. Если операционная система запросит разрешение на установку приложений из внешних источников (браузера или проводника), разрешите установку.
5. Запустите приложение и введите ваше имя для старта!

## 🔄 Система обновлений

Начиная с версии v1.0.3, в приложении доступна ручная проверка обновлений:
1. Перейдите в раздел **Настройки** ➔ **Проверить обновления**.
2. При наличии более новой сборки приложение покажет список изменений и предложит скачать свежий установочный файл напрямую с этой страницы.
